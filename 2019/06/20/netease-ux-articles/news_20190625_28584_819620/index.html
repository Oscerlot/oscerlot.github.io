Vim Application Background

The full name of the Vim editor is Visual interface improved, and the specific interface is shown in the figure below.

The application scenarios of the Vim editor mainly include:

(1) Need to deploy the environment and develop code in a Linux environment. Linux often requires the use of the VIM editor for code development;

Currently, many IDEs support VIM mode, such as PyCharm/IntelliJ IDEA and other mainstream IDEs. The advantage of the Vim editor is that it does not require much mouse operation and can support all operations using the keyboard, achieving fast movement, fast input, and effective block operations. The disadvantage of the Vim editor is that it requires memorizing many operation shortcuts and command lines.

4 Modes of Vim Editor

The Vim editor has four modes. It is in normal mode by default, including normal mode, insert mode, command-line mode, and visual mode. The conversion relationship between the 4 modes is shown in the figure below.

So how are the 4 modes converted?

(1) Convert normal mode to insert mode, the specific operations are as follows:

Command Description
i Insert text before the cursor
I Insert text at the beginning of the line
a Append text after the cursor
A Append text at the end of the line
o Open a new line below the current line and enter insert mode
O Open a new line above the current line and enter insert mode
ea Append text at the end of the word

(2) Convert insert mode to normal mode, the specific operations are as follows:

Command Description
Esc Switch back to Normal mode from Insert mode.
Ctrl-[ An alternative that achieves the same as ESC.
Ctrl-C Same effect

(3) Convert normal mode to command-line mode, the specific operations are as follows:

Command Description
: Enter command-line mode. You then type a command and press Enter.
/ To search forward.
? To search backwards.

(4) Convert command-line mode to normal mode, the specific operations are as follows:

Command Description
Esc Return to Normal mode.
Enter Execute the command and return to Normal Mode.
Ctrl-[ Return to Normal Mode.
Ctrl-C Return to Normal Mode.

(5) Convert normal mode to visual mode, the specific operations are as follows:

Command Description
v Enter visual mode (characterwise). You can then move the cursor to select text.
V Enter visual line mode (linewise). Selects entire lines.
Ctrl-v Enter visual block mode (rectangular selection). Very useful for selecting columns of text.

(6) Convert visual mode to normal mode, the specific operations are as follows:

Command Description
Esc Return to normal mode
Ctrl-[ Return to Normal Mode.
Ctrl-C Return to Normal Mode.

Vim Operation: Cursor Movement

Vim editor cursor movement is all operated in normal mode.

(1) Unit level: character as the unit

Command Description
h Move left one character
j Move down one line
k Move up one line
l Move right one character

(2) Word level

Command Description
w or W Move to the beginning of the next word
e or E Move to the end of the current word
b or B Move to the beginning of the previous word

(3) Block level

Command Description
gg Go to the first line of the document
G Go to the last line of the document
0 Go to the beginning of the line (column 1)
^ Go to the first non-blank character of the line
$ Go to the end of the line
Ctrl-d Move down half a page
Ctrl-u Move up half a page
Ctrl-f Move down one page
Ctrl-b Move up one page
:<N> or <N>gg Jump to line N
:+<N> or <N>j Jump down N lines
:-<N> or <N>k Jump up N lines

It should be noted here that: all commands can be preceded by a number N, indicating that the following command is executed N times. For example, if you want to move down 3 lines, in addition to using :+3, you can also use 3j to achieve the same effect. In addition, there are actually two types of commands above: one is executed immediately after typing, such as gg; the other is to enter : first (there will also be a slash / first later). This type of command needs to be executed by pressing Enter after input, and the same is true in the following tutorial content.

Vim Operation: Open File, Find Content

Vim editor file and content operations are also performed in normal mode.

(1) Open a file in Vim

Command Description
:e <filename> Open the file named filename, if the file does not exist, create it
:Ex Open the directory tree in Vim, select with the cursor and press Enter to open the corresponding file (Tip: - enter the upper-level directory)

(2) Find within the document

Command Description
* Find the next occurrence of the word currently under the cursor.
# Find the previous occurrence of the word currently under the cursor.
/<search> Search forward for the specified string.
?<search> Search backward for the specified string.
n Continue searching for the next occurrence.
N Continue searching for the previous occurrence.

Note for the above operations: n and N are directional. If you searched by * before, n will continue to search towards the end of the document, and N will search towards the beginning of the document; conversely, if you searched by #, n will point to the beginning of the document, and N will point to the end of the document.

Vim Operation: File Modification and Saving

(1) Insert: in insert mode

Command Description
i Insert before current char
a Insert after the current character
I Insert at the beginning of the line
A Insert at the end of the line
o Insert on the next line
O Insert on the previous line

Precautions for file insertion operations: Any of the above commands will make Vim enter insert mode. After entering insert mode, the cursor will change. At this time, the text entered will appear directly in the document. Press the Esc key or Ctrl-[ or Ctrl-C to exit insert mode

(2) Delete (and save to Vim clipboard): in normal mode

Command Description
x Delete the current character, equivalent to Delete in insert mode
X Delete the previous character, equivalent to Backspace in insert mode
dd Delete the current line and save the deleted content to the Vim clipboard
d<X> Delete the specified content and save it to the Vim clipboard
c<X> Delete the specified content, save it to the Vim clipboard, and enter insert mode

Supplementary note for delete operation: The part is a description of the operation content. To delete a word, enter dw or de; to copy the content from the current position to the end of the line, enter y$; to delete the next 3 characters and insert, enter c3l, etc.

(3) Copy: in normal mode

Command Description
yy Copy the current line to the Vim clipboard
y<X> Copy the specified content to the Vim clipboard

(4) Paste: in normal mode

Command Description
p Paste after the current position
P Paste before the current position

(5) Replace: in normal mode

Command Description
r<X> Replace the current character with X
gu<X> Convert the specified text to lowercase
gU<X> Convert the specified text to uppercase

In addition to the above replacement operations in normal mode, you can use the commands in command-line mode to replace the content of the document. The specific command format is as follows:

:[range]s/search/string/[flags] [count]

Search for the search content according to the pattern regular expression, and replace it with the string replacement content. The description of the options in the specific command is shown in the table below.

Option Description
range The range of lines. % means the entire file, same as 1,$. . means current line and $ the last line.
search The text to search for
string The text to replace search with.
flags g (global) replaces all occurrences. c (confirm) requires confirmation before each replacement.
count The number of lines to apply the command to (starting from the end of range or the current line).

(6) Undo, redo: in normal mode

Command Description
u Undo
Ctrl-r Redo

(7) Save the file: in command-line mode

Command Description
:w Save the current file
:wa Save all files
:wq or ZZ Save and exit
:q! or ZQ Force quit, do not save
:saveas <newfilename> Save the file as
:w <newfilename> Save a copy of the file as and continue editing the original file

Vim Settings and Tips

(1) Vim settings

Command Description
:set number Show line numbers
:set relativenumber Show relative line numbers (this is very important, take your time to appreciate)
:set hlsearch Highlight search results
:set autoindent Automatic indentation
:set smartindent Smart indentation
:set tabstop=4 Set the width of the tab character to 4
:set softtabstop=4 Set the indent width to 4 when pressing tab
:set shiftwidth=4 Set the automatic indentation width to 4
:set expandtab Convert tab characters to spaces when indenting
:filetype on Enable file type detection
:syntax on Enable syntax highlighting

Supplementary note: The commands are listed here. You can enter them in Vim for setting, but the parameters set in this way are only effective before closing Vim this time. If you exit Vim and open it again, the previous settings will be invalid. To make it permanently effective, you need to modify an automatic configuration file of Vim, generally the file path is /home//.vimrc (Linux system) or /Users//.vimrc (MacOS system)

(2) Repeat the last command: in normal mode

Vim has a special command: .

(3) Indentation: in normal mode

Command Description
>> Indent the current line to the right
<< Indent the current line to the left

Try pressing 3>>, <<, and <G on this line.

(4) Automatic formatting: in normal mode

Command Description
== Automatically format the current line
gg=G Automatically format the entire current document
<N>== Automatically format N lines starting from the current line
=<N>j Automatically format the current line and N lines down
=<N>k Automatically format the current line and N lines up

(5) Split screen: in command-line mode

Command Description
:split (abbr: :sp) or Ctrl-w s Split horizontally
:vsplit (abbr: :vs) or Ctrl-w v Split vertically

(6) Window jump

Command Description
Ctrl-w w Activate next window
Ctrl-w j Activate window below
Ctrl-w k Activate window above
Ctrl-w h Activate window left
Ctrl-w l Activate window right

(7) Screen zoom

Command Description
Ctrl-w = Average window size
Ctrl-w + Increase height
Ctrl-w - Decrease height
Ctrl-w _ Maximum height
Ctrl-w > Increase width
Ctrl-w < Reduce width
Ctrl-w | Maximum width
