Introduction to Sublime

Sublime Text is a code editor with a beautiful user interface and powerful features, such as code thumbnails, Python plugins, code snippets, etc. Key bindings, menus, and toolbars are also customizable. Sublime Text’s main features include spell check, bookmarks, a complete Python API, Goto functionality, instant project switching, multiple selections, multiple windows, and more. Sublime Text is a cross-platform editor that supports Windows, Linux, Mac OS X, and other operating systems.

Case Demonstration

Sublime was once the mainstream front-end editor (although it has gradually been replaced by VSCode now), and its powerful text functions are also commendable for the efficiency of big data development. Next, an example will be introduced to help everyone appreciate the brilliance of Sublime:

Sublime supports the syntax of most mainstream programming languages. When we open a piece of text, we only need to set it to the corresponding syntax format in the View –> Syntax of the main menu to highlight the keywords:

 

One of the highlights of Sublime Text is that it supports multiple selections - selecting multiple areas at the same time, and then editing them at the same time.

Ctrl + D selects the word where the cursor is currently located and highlights all occurrences of the word. Ctrl + D again selects the next occurrence of the word. During the multiple word selection process, use Ctrl + K to skip, Ctrl + U to go back, and Esc to exit multiple editing.

A major application scenario of multiple word selection is renaming - to make the code cleaner. Although Sublime Text cannot automatically rename like an IDE (such as Eclipse), we can use multiple selection + multiple editing for intuitive and convenient renaming:

 

In programming languages that use semicolons as statement terminators, the mixed use of special characters seems to have little effect on the compilation and operation of the code, but in languages like Python that use indentation as syntax, mixing tabs and spaces will cause errors when the code is executed. Sublime’s lower right corner can set Tab Size: how many spaces a Tab key is converted into, and then select Convert Indentation To Spaces to convert all Tab keys to space keys.

Finding & Replacing

Sublime Text provides powerful find (and replace) functions. The basis of replace is find, so the replace function is introduced here. Use Ctrl + H (Command + option + F on Mac) to bring up the standard find & replace module. The first button on the left of the first row is the regular expression switch, which means whether the searched content is a regular expression or an ordinary string. Summary

In addition to the built-in functions of the editor, Sublime can also be used as a simple compiler (the compiler call path needs to be configured by yourself). You can install the package control console (Package Control), use Ctrl+Shift+P (Command+Shift+P on Mac) to call it up, and then you can install and use various extension plugins on Sublime.
