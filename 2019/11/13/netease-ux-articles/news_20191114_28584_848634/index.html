After data exploration and data preprocessing, we obtain data that can be directly modeled. According to the mining objectives and data format, we can establish models such as classification and prediction, cluster analysis, association rules, time series patterns, and deviation detection.

Classification and Prediction

Classification and prediction are two main types of prediction problems. Classification mainly predicts class labels (discrete attributes), while prediction mainly establishes continuous value function models to predict the value of the dependent variable corresponding to a given independent variable.

1.1 Implementation Process

(1) Classification is to construct a classification model, input the attribute values of the sample, and output the corresponding category, mapping each sample to a predefined category.

The classification model is also established on the existing labeled data set, and the accuracy of the model on the existing samples can be easily calculated, which belongs to supervised learning.

(2) Prediction refers to establishing a functional model of interdependence between two or more variables, and then making predictions or controls.

(3) Implementation process

The first step of classification is the learning step, which establishes a classification model to obtain classification rules through inductive analysis of the training sample set. The second step is the classification step, which first uses a known test sample set to evaluate the accuracy of the classification rules and make predictions.

The first step of prediction is to establish a functional model of predictive attributes through the training set. The second step is to perform prediction or control after the model has passed the test.

1.2 Common Classification and Prediction Algorithms

① Regression analysis: Regression analysis is the most common statistical method to determine the quantitative relationship between predictive attributes (numerical) and other variables. It includes linear and nonlinear regression, Logistic regression, ridge regression, principal component regression, partial least squares regression, etc.

② Decision tree: Adopts a top-down recursive method, compares attribute values at internal nodes, and branches down from the node according to different attribute values. The leaf node finally obtained is the class of learning division.

③ Artificial neural network: An artificial neural network is an information processing system established by imitating the structure and function of the brain’s neural network. It is a model that represents the relationship between the input and output variables of the neural network.

④ Bayesian network: Bayesian network, also known as belief network, is an extension of the Bayes method and is currently one of the most effective theoretical models in the field of uncertain knowledge representation and reasoning.

⑤ Support vector machine: Support vector machine is an algorithm that converts low-dimensional nonlinear separability into high-dimensional linear separability through a certain nonlinear mapping, and performs linear analysis in high-dimensional space.

1.3 Regression Analysis

(1) Research scope of regression analysis

Problem Type Description Example
Simple Linear Regression Relationship between one independent variable and one dependent variable. Predicting sales based on advertising spending.
Multiple Linear Regression Relationship between multiple independent variables and one dependent variable. Predicting house prices based on size, location, and number of bedrooms.
Nonlinear Regression Relationship where the dependent variable is not a linear function of the independent variables. Predicting population growth based on time.
Logistic Regression Predicting the probability of a binary outcome (0 or 1). Predicting whether a customer will click on an ad (yes/no).

(2) Commonly Used Models for Regression Analysis

Model Formula Description
Linear Regression Y = β0 + β1X + ε Predicts a continuous dependent variable Y based on a single independent variable X.
Multiple Linear Regression Y = β0 + β1X1 + β2X2 + ... + βnXn + ε Predicts a continuous dependent variable Y based on multiple independent variables X1, X2, ... Xn.
Logistic Regression log(p/(1-p)) = β0 + β1X Predicts the probability p of a binary outcome (0 or 1) based on a single independent variable X.

Note: Linear regression models are usually relatively simple regression models, but the independent and dependent variables can have some form of curved relationship, requiring the creation of a nonlinear regression model.

① Logistic Regression Analysis Introduction

Logistic function: In the logistic regression model, the dependent variable has only two values, 0 and 1. The logarithm is taken after taking the event ratio, and the function is obtained:

log(p/(1-p)) = β0 + β1X1 + β2X2 +...+ βpXp

② Modeling Steps: First, set indicator variables according to the analysis objectives, then collect data and screen features; list the regression equation and estimate the regression coefficients in the model; perform model testing, including accuracy, confusion matrix, ROC curve, and KS value; model application, input the value of the independent variable, you can get the value of the predicted variable, or control the value of the independent variable according to the value of the predicted variable.

For example: Logistic regression modeling based on the data of a certain bank in reducing loan delinquency rate.
