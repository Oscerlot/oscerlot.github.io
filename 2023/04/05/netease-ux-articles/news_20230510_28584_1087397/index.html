The Interactive Art of Loading in Games
5 Apr, 2023

Machine Translated from Chinese to English
Original article written by ThunderFire UX (NetEase)

 

Game loading design has undergone iterations and innovations along with the development of the game industry. It can even be cleverly designed in combination with game style and story, bringing players a better sense of immersion and a more interesting gaming experience. This article will introduce the development path of game loading, focus on the design methods of game loading, and analyze the design art of game loading from the two aspects of strong perception and weak perception of game loading.

1. The Path of Development: From Appearance to Elimination

The Original Meaning - “Reducing Anxiety”

Looking back on the time when playing Famicom games, you will find that you don’t seem to have any memory of the game loading screen. This is because the game capacity at that time was small, the storage medium was a cartridge, and the reading speed was fast. The game could basically run without loading.

In the 1980s, the storage medium for games became magnetic tape. The popular game equipment was a home computer equipped with an 8-bit microprocessor and 64k of memory. The computer read data according to the rotation of the magnetic tape. The loading time of more than ten minutes would inevitably lead to misunderstanding and anxiety about “computer crashed?” if only a black screen was displayed to the player. So the text loading prompt appeared, but this simple transition from static to dynamic form could only barely pass the boring time of loading.

 

Compared with the monotonous text prompts, many games chose to use original paintings as loading screens to help players pass the time. For the art resources that were suppressed by program restrictions at that time, this was a rare opportunity for “high-definition” display. Although the loading time was extended by 10% due to the call of art materials, this also became a rare opportunity for players to see “high-definition” pictures, so it was very popular.

 

Game Screen (left) and Loading Screen (right)

By the end of the 1980s, progress bars had already begun to be used in games. Although the first game to design a progress bar is no longer testable, this design that can intuitively reduce the player’s waiting anxiety is still not outdated.

Pioneering Design - “Game in Game”

With the upgrade of software and hardware, the game reading speed is faster. In Namco’s Ridge Racer in 1993, the game changed the previous loading form and placed the playable mini-game Galaxian in it, making the 30-second loading time pass abnormally fast, and the response in the player community was very good.

 

Loading game “Galaxian” in “Ridge Racer”

Although Namco applied for a patent for the mini-game in the loading process - “auxiliary game”, which created a shackle for its creation, the mini-game in this patent and the main game are two separate games. Later generations cleverly avoided this point and loaded part of the main game code to make it a mini-game. Part of the main game content was migrated and designed into an interactive game loading, which achieved the design of “game in game” and also made the way of game loading enter a more distinctive new stage.

Unremitting Pursuit - “Eliminate” Loading

Whether it is a static prompt, a dynamic picture, or an interactive mini-game, these are just to distract the player’s attention. However, the player knows that it is loading, and the game experience is inevitably affected. “Elimination” is the optimal solution that does not affect the game experience and concentration, and this is also the unremitting pursuit of many game loading designs.

In 2005, Tony Hawk’s American Wasteland directly used the no-loading screen as a selling point of the game, and players could experience the thrill of seamless exploration. It connects the game map through passages. When players enter the enclosed space of the passage, the map is loaded, allowing players to complete the loading unknowingly while experiencing the game.

 

2. A Thousand Producers Have a Thousand Loading Design Methods

The player experience in the game will inevitably be affected by memory reading, network speed, and response speed. Players often cannot get the feedback they want immediately after operating, and always have to wait for a while.

The Doherty Threshold theory points out that when the user’s command is executed by the system and the answer is returned within 400 milliseconds, it can keep them focused, improve productivity, and improve the negative experience of waiting. Extending the Doherty Threshold theory to the application of game loading design methods, it can be found that giving players timely feedback during game loading and the perceptibility of the waiting feedback process can maintain the player’s concentration and provide a good game experience.

The strong perception of game loading emphasizes the visualization of the feedback loading process and clarifies the waiting progress. The weak perception of the game loading process emphasizes the timeliness of feedback, so that players are unconsciously in the process of waiting.

Game Loading Design with Strong Perception

In the game loading design that players can clearly perceive, regardless of its accuracy, the visual form can well reduce their waiting anxiety. The flexible use of loading animations, progress bars, splash pages, and art resources are commonly used design methods.

● Progress Bar

Animation is a good way to attract players visually, and the progress bar is the best form of waiting time visualization. The progress bar can be designed in combination with elements that reflect acceleration, connection, etc. in the game, and with meaningful animation effects, it can give more vitality to the functional loading form. For example, in Onmyoji, the loading progress bar is displayed by the Shikigami Usagi, who has the attribute of accelerating assistance, riding a magic frog and jumping forward, which is interesting and vivid and has the meaning of the game world view.

 

● Splash Page

The splash page can not only bring a unique transition effect, but also has a fascinating guiding effect. Common forms include opening doors, closing fans, turning pages, notes, and so on. In Red Dead Redemption, the splash screen and fading animation are combined with black and white photos of the scenery, which directly brings people’s thoughts into the western world of the twentieth century.

 

Loading display of “Red Dead Redemption”

● Art Resources

Art resources are also the most commonly used content for showing off skills in game loading, which can increase players’ perceived value of the game style and establish their trust in the game. For example, the exquisite original paintings allow players to appreciate the exquisite production of game art, and also bring players a relaxed and pleasant waiting experience; the display of game models during loading, and even some free interactions, can bring players a unique experience different from the rhythm of the game.

 

Loading display of “The Elder Scrolls V”

The loading form of excellent art resources can also help the connection of the game plot. For example, the loading screen of The Witcher 3 is a plot introduction, which helps players recall the main line and also brings a movie-level look and feel.

 

Loading display of “The Witcher 3”

Game Loading Design with Weak Perception

The goal of weakly perceived loading design is to allow players to spend the game loading time unknowingly, by cleverly using “blinders” to naturally turn the loading screen into a part of the game.

● Passage

In the game’s elevators, cable cars, or winding aisles, the player’s forward speed will be limited, thus giving the game ample loading time. The design of the passage is widely used in the game loading design of weak perception. For example, there are always many elevators in Hidetaka Miyazaki’s games, and the game completes the loading of switching scene resources while the player is riding the elevator. In Souls-like games, two large scenes are usually connected by a narrow cave or an empty dungeon, hiding the loading process outside the narrow space. In God of War III, players control the characters to fly in the tunnel to avoid falling rocks, perfectly combining the passage with interactive loading.

 

● Slow Walking

Slow walking is a loading form that is more difficult for players to detect. The slowed footsteps before entering the plot, and even when the game is preloading subsequent resources, in order to prevent the player from walking too fast, there will be settings such as swamps, narrow gaps that can only be passed slowly, or deliberately tripping the character, etc. to delay time for loading. These are the means used by the production team to use actions and terrain to make the game loading and game content more naturally connected.

 

Slow walking loading of “God of War 4”

● Camera Movement

In fact, it is not difficult to perfectly hide the loading process in the game. The game only needs to limit the player’s perspective through the camera to easily hide the loading process from view. As early as the PS1 era, there were applications of using the camera to cover up loading, such as in Spyro the Dragon, as Spyro flies, the game perspective changes from the ground to the sky, hiding the loading process of ground resources. In Hellblade: Senua’s Sacrifice, the close-up shot is used to express the heroine’s inner activities and follow her perspective to complete the scene switching.

 

Close-up loading in “Hellblade: Senua’s Sacrifice”

3. Summary

Today, the design of game loading is not limited to specific routines or blindly hiding. Instead, it is constantly exploring the perfect integration of game loading with game content and worldview. More and more games are carefully designed between maps, and even no trace of loading can be seen.

Due to the technical development of memory reading speed, the current game loading process cannot be truly eliminated. While technology is sprinting forward to accelerate loading, the content design of game loading has become an art. It can be cleverly designed in combination with the game style and story narration, which can not only help the game improve the worldview, but also optimize the user experience, increase retention, and bring players a better sense of immersion and a more interesting gaming experience.
